# WEB GOIKO PROYECTO

Proyecto web inspirado en la cadena de restaurantes Goiko, desarrollado con HTML, CSS y JavaScript. El objetivo de este proyecto es recrear una página web moderna, visualmente atractiva y totalmente responsive, aplicando conceptos fundamentales del desarrollo web front-end.

## Descripción

La web presenta información sobre el restaurante, sus productos y servicios mediante una interfaz intuitiva y una estructura organizada. El proyecto ha sido diseñado para ofrecer una buena experiencia de usuario tanto en dispositivos de escritorio como móviles.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Diseño responsive adaptable a diferentes tamaños de pantalla.
* Navegación sencilla e intuitiva.
* Elementos interactivos desarrollados con JavaScript.
* Estilos personalizados mediante CSS.
