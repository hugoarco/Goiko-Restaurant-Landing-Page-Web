<h1 align = "center" >Goiko Restaurant – Landing Page Web </h1> 

Proyecto de desarrollo web front-end inspirado en la cadena de restauración Goiko, diseñado como una landing page moderna, responsive y orientada a experiencia de usuario.

El objetivo del proyecto es aplicar conceptos fundamentales de desarrollo web moderno, combinando estructura, diseño visual e interactividad.

🎯 Objetivo del proyecto
<br><br>
Desarrollar una página web atractiva y funcional que simule la presencia online de un restaurante real, aplicando buenas prácticas de:
Diseño responsive
Estructura semántica HTML
Estilado avanzado con CSS
Interactividad básica con JavaScript
<br><br>
🧱 Tecnologías utilizadas
HTML5 (estructura semántica)
CSS3 (diseño y estilos visuales)
JavaScript (interactividad y dinámicas básicas)
<br><br>

✨ Funcionalidades
<br><br>
📱 Diseño totalmente responsive adaptado a móvil, tablet y escritorio
<br><br>
🧭 Navegación intuitiva y fluida
<br><br>
🎨 Diseño visual moderno inspirado en branding de restauración
<br><br>
⚡ Interactividad básica mediante JavaScript
<br><br>
🧩 Estructura modular de secciones (menú, presentación, información, etc.)
<br><br>
